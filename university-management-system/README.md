# University Management System

A comprehensive university management API built with FastAPI and MySQL.

## Features

- ✅ User authentication (JWT)
- ✅ Role-based access (Student, Teacher, Admin)
- ✅ Schedule management (by group/teacher/auditorium)
- ✅ Grade tracking with debt identification
- ✅ Applications (certificates, retakes, scholarships)
- ✅ File uploads (PDF, DOC, JPG) with 250 MB limit
- ✅ Announcements system
- ✅ Email simulation for password generation

## Tech Stack

- **Python 3.9+**
- **FastAPI** - Web framework
- **SQLAlchemy** - ORM
- **MySQL** - Database
- **JWT** - Authentication
- **BCrypt** - Password hashing
